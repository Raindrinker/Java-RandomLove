# Java-RandomLove
A library made to use and abuse randomness in Java, with some data structures and whatnot. Designed for games, but can be used for anything.

###LIST OF STUFFS

* **Pool**

Very basic data structure where to store objects of any type to later select one of them randomly.

* **WeightedPool**

Version of the Pool where each one of the objects in it has a weight value (implement Weighted). When randomly selecting one of the objects in the pool, the chance of each object being selected is proportional to its weight value.
